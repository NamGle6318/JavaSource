인터페이스
- 추상클래스(단 추상화 정도가 높다)
- 일반메서드, 멤버 변수를 가질 수 없음 ( 상수는 선언 가능)
- 단 일반 메서드 앞에 static이나 default가 붙는 경우 선언 가능 (java 1.8 버전 이후로 가능)
- 상수밖에 선언할 수 없기 때문에 선언시 public static final 생략 가능
- 추상메서드만 선언 가능하기 때문에 abstract 을 생략할 수 있다.
- 인터페이스 클래스의 구성 (상수, 추상메서드)
- 인스턴스 직접 생성 불가능 (new)
- 인터페이스 상속 : - 인터페이스로만 상속을 받는다. / 다중상속
- 장점 : 표준화가 가능하다 / 서로 관계없는 클래스들에게 관계를 맺어 줄 수 있다.
