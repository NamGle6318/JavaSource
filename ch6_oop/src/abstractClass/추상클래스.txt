추상 클래스 (abstract)
- 추상화 : 클래스간의 공통점을 찾아내서 공통의 조상을 만드는 작업
            ex : 공통된 멤버변수나 메서드를 찾아서 상위 클래스로 만들어내기
- 구체화 : 상속을 통해 클래스를 구현, 확장하는 작업
            ex : 추상화해서 추려낸걸 클래스로 만들기 
- 추상 메서드 사용 
- 어떤 클래스 내에 추상 메서드가 하나라도 있으면 클래스는 무조건 추상 클래스여야 함.
- 인스턴스 생성 불가능(!= new)
- 부모 클래스의 역할
- 추상클래스와 상속관계인 하위 클래스들은 조상클래스의 추상 메서드 구현을 강제함.
